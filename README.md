## Project Goals
 - Experiment with FastAPI
 - Build an API in Python using async-first tools.
   - This includes aiohttp instead of requests
   - This includes Tortoise ORM
 - Build auth flow(s)
 - UserModel in Tortoise
 - Password hashing with passlib
 - generating API keys

## Order of Operations for Authentication

1. Create User
2. Generate Token
3. Use Token to Access Protected Routes
